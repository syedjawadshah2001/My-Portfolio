Automated Document Format Checking System

This final year project is a web-based automation platform designed to reduce manual document formatting review. The system validates uploaded academic documents against defined formatting rules, including margins, fonts, spacing, alignment, and structural layout. It uses Python and Flask for application logic, PDF parsing libraries for document inspection, and Firebase Cloud Firestore for real-time document tracking and storage.

The platform was successfully deployed on Streamlit Community Cloud and achieved 100% validation accuracy under the rule-based testing scenarios defined for the project. Its main value is replacing slow manual checking with consistent automated validation.
