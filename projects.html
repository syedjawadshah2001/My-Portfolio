Python · Flask · PDF Parsing · Firebase · Streamlit

Automated Document Format Checking System

A web-based automation platform that eliminates manual academic document formatting review. The system accepts PDF uploads and validates them against a defined rule set — checking margins, font types and sizes, line spacing, text alignment, heading hierarchy, and structural layout. Every rule produces a pass/fail result with a precise location reference.

Built with Python and Flask for the application layer, PDF parsing libraries for document inspection, and Firebase Cloud Firestore for real-time document tracking and result storage. Deployed on Streamlit Community Cloud for public access.

Outcome: 100% rule-validation accuracy across all test scenarios defined for the project evaluation. Replaced manual checking with consistent, instant automated feedback.

Automated Book Generation System

An AI-powered backend system that generates structured, multi-chapter book content from a user-supplied topic. FastAPI handles the API layer, OpenAI generates section and chapter content from carefully constructed prompts, and Supabase (PostgreSQL) stores generated books with full authentication and user management support.

The architecture demonstrates end-to-end AI product design: input collection, prompt construction, content generation, structured storage, and export preparation into document formats such as PDF and DOCX. The system is built to be extended — adding new content types, export formats, or user tier management requires minimal structural change.
